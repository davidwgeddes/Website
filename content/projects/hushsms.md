A simple, text-only private messaging app for Android that brings military-grade encryption to your everyday SMS communications.

## Features

- **End-to-End Encryption** - All messages are encrypted using industry-standard algorithms
- **Text-Only Design** - Focused, minimal interface for pure messaging
- **Native SMS Integration** - Works with Android's built-in SMS facilities
- **No Account Required** - No registration, login, or online account needed
- **Privacy First** - No data collection, no ads, no tracking
- **Message Self-Destruct** - Optional automatic message deletion
- **Contact Management** - Secure your contacts with encryption
- **Open Source** - Transparent code for security auditing

## Tech Stack

- Android native development
- End-to-end encryption (AES-256)
- SMS-based communication
- No external servers required
